Selain untuk menampilkan data, tabel pada HTML dapat digunakan untuk membuat desain layout atau tata letak sebuah halaman web. Untuk mengaplikasikan tabel sebagai layout sebuah halaman web, perlu ditambahkan atribut role dengan value presentation pada tag <table>, atau ubah nilai atribut border menjadi 0, seperti berikut:


Berikut ini adalah contoh desain layout yang umum diterapkan pada sebuah halaman web beserta penjelasannya:

1. Header

Menampilkan identitas utama dari sebuah halaman web yang diakses. Biasanya berisi logo dan nama website. Letak dari header selalu di bagian paling atas dari sebuah halaman web.

2. Navigation bar

Menyediakan navigasi yang mempermudah user untuk menuju ke halaman-halaman lain yang dapat diakses.

3. Sidebar

Berisi daftar konten-konten lain, iklan, atau menu tambahan, dan terletak di samping kiri, kanan, atau keduanya.

4. Content

Berisi konten utama yang akan ditampilkan sebuah halaman web kepada user.

5. Footer

Bagian yang terletak paling bawah dan menampilkan informasi dari sebuah website. Informasi-informasi tersebut di antaranya pembuat website, copyright, tahun pembuatan, lisensi penggunaan, contact, dan lain-lain.
