Apa Itu HTML??


 HTML merupakan singkatan dari HyperText Markup Language adalah script untuk menyusun dokumen-dokumen web. Mendesain HTML berarti malakukan suatu tindakan pemrograman. HTML bukanlah sebuah bahasa pemrograman melainkanperintah-perintah yang telah terstruktur berupa tag-tag penyusun. Dengan HTML, kita dapat memberi, Teks, Musik, Video, pada halaman web. dengan memasukkan script tertentu. Untuk Menampilkan konten-konten tersebut, diperlukan memasukkan kode-kode perintah atau script tertentu yang disebut tag atau kode perintah. Selain Tag, terdapat pula istilah-istilah lain seperti element dan atribut.

Tag, elemen, dan atribut

 Tag pada HTML merupakan sebuah perintah yang digunakan untuk menampilkan konten pada web. Tag pada HTML selalu memiliki kode perintah pembuka dan penutup. Tag ditulis dengan format nama tag yang dikelilingi oleh tanda "<" dan tanda ">" pada pembuka dan diawali tanda "/" setelah tanda "<" pada penutupnya.
Berikut adalah format penulisan tag:



 Beberapa macam tag HTML yang umum digunakan dapat berupa tag <p> yang berguna untuk menampilkan paragraf, tag <h1> untuk menampilkan caption dengan ukuran tertentu, tag <h1> memiliki ukuran , h1 merupakan tag untuk menunjukkan caption terbesar sedangkan h6 merupakan tag untuk menunjukkan caption terkecil, tag <img> untuk menampilkan gambar, tag <table> untuk membuat tabel, dan masih banyak lagi.

  Elemen merupakan isi dari sebuah tag atau sebua yang terletak di antara tag pembuka dan tag penutup. sedangkan Atribut adalah informasi tambahan yang diberikan kepada tag yang berfungsi untuk memberi efek tambahan dari tag tersebut dan biasanya ditambahkan di tag pembuka saja. contoh, atribut style yang ditambahkan pada tag pembuka <p> untuk menambah gaya atau bentuk teks elemen yang ada di dalam tag <p> tersebut.

 <!DOCTYPE html> ditambahkan di awal sebagai deklarasi untuk memberitahu browser bahwa dokumen/halaman web tersebut merupakan HTML versi HTML5. Jika deklarasi ini tidak ditambahkan maka browser akan melihat dokumen web tersebut sebagai dokumen/halaman web dengan versi yang telah usang.

  tag <html> digunakan setelahnya untuk menyatakan bahwa script/dokumen yang digunakan adalah script/dokumen HTML. Begitu pula penutupnya </html> digunakan pada paling akhir script tersebut.

  tag <head> biasa digunakan sebagai caption/judul dan biasanya digunakan sebagai tempat menaruh script css. penutupnya <head> digunakan jika telah memasukkan caption atau style tertentu.

  tag <body> digunakan untuk memasukkan script html yang akan di tampilkan di halaman web. seperti tag <p>, <h1>, <table>, dll.
